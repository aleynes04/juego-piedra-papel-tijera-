#  Proyecto: Piedra, Papel o Tijera - Cyber Yellow

Este es un juego clásico desarrollado con tecnologías web modernas, inspirado en una estética industrial y futurista. El jugador se enfrenta a la computadora en un duelo al mejor de 3 rondas.

##  Requisitos
Solo necesitas un navegador web actualizado (Chrome, Edge, Firefox, etc.).

##  Instalación y Ejecución
1. Descarga los archivos `index.html`, `style.css` y `script.js`.
2. Guárdalos todos en la **misma carpeta**.
3. Haz doble clic en `index.html` para iniciar el juego.

##  Instrucciones de Juego
- Selecciona una de las tres opciones: **Piedra, Papel o Tijera**.
- La **Computadora** elegirá una opción al azar simultáneamente.
- El primer jugador en alcanzar **3 victorias** será el ganador del encuentro.
- Puedes reiniciar el marcador en cualquier momento con el botón "REINICIAR SISTEMA".

##  Tecnologías
- HTML5 / CSS3 (Variables, Flexbox, Gradients).
- JavaScript Vanilla (Manipulación del DOM y Lógica Aleatoria).
