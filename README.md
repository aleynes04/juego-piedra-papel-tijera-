# ⚡ SISTEMA DE COMBATE: Piedra, Papel o Tijera (v1.0) ⚡

¡Bienvenido a la primera versión del **Sistema de Combate**! Una experiencia interactiva, dinámica y directa para disfrutar del clásico juego *Piedra, Papel o Tijera*, diseñada con una fuerte estética inmersiva y tecnología web nativa. 🚀

## ✨ Características de Lanzamiento

En esta **Versión 1.0**, el proyecto nace con una base sólida, enfocada en la experiencia del usuario y un rendimiento limpio:

- 🎨 **Interfaz Estilo Cyberpunk:** Un diseño oscuro con acentos amarillos vibrantes (`#ffdf00`) y bordes dinámicos que se iluminan (verde o rojo) según el resultado de tu combate.
- 🎵 **Audio Integrado Nativo:** ¡El juego suena sin archivos mp3 pesados! Usamos la Web Audio API para generar tonos retro/8-bits matemáticamente cuando haces clics, ganas o pierdes.
- 📳 **Game Feel:** Pequeñas animaciones de *shake* (sacudida) que añaden ese toque extra de suspenso mientras el adversario (PC) "sintetiza" su decisión.
- 💾 **Historial Permanente:** Un registro local básico (`localStorage`) que mantiene tu cuenta global de victorias 🏆 y derrotas aunque cierres y abras la pestaña.
- ⚙️ **Configuración de Rondas:** Podrás elegir rápidamente si deseas enfrentarte al mejor de 1, 3 o 5 rondas.

## 🛠️ Tecnologías del Proyecto

- **HTML5:** Marcado semántico, meta-etiquetas amigables (`aria-label`) para lectura de accesibilidad.
- **CSS3:** Sistema de Layouts flexible, animaciones puras con `@keyframes` y un diseño 100% responsivo para móviles.
- **JavaScript (POO):** Código base Orientado a Objetos (Clases de ES6) para facilitar la lectura, el mantenimiento y futuras incorporaciones de más funcionalidades.

## 🕹️ Cómo Empezar

1. No necesitas instalar nada. Solo ve a la carpeta del proyecto y ejecuta doble clic en el archivo `index.html`.
2. En la barra superior, ajusta tu meta (Ej. *Mejor de 3 Puntos*).
3. Presiona sobre **Piedra ✊**, **Papel 📄** o **Tijera ✂️** para enfrentar al sistema de la computadora.
4. Intenta ganar tantas partidas como te sea posible para mejorar tus estadísticas históricas.
